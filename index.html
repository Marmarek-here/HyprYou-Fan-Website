Overview

What HyprYou adds on top of raw Hyprland

deployed_code

Designed around Material You

HyprYou follows the Material 3 language closely: rounded surfaces, tonal color systems, large motion-driven components, and UI pieces that feel consistent instead of improvised.

Wallpaper-aware color generation

Colors can be generated from wallpaper or settings, then applied across apps and interface layers, which is the main reason the desktop feels cohesive instead of theme-stacked.

Why people use it

  • Large parts of the setup are controlled from settings instead of scattered config edits.
  • Optional package layers let you add dialogs and greeter support when you need them.
  • It keeps a cleaner home-directory footprint than many hand-rolled dotfile setups.

Session detail

HyprYou can run as its own desktop session.

If you launch it from a display manager, use the HyprYou session rather than plain Hyprland. Custom variables and overrides live in ~/.config/hypryou/hyprland.conf.

Three ways to install HyprYou on Arch

Method 1: AUR helper

The fastest route if you already use yay, paru, or another AUR helper.

yay -Ss hypryou
paru -Ss hypryou

yay -S hypryou-git
paru -S hypryou-git
# optional
# yay -S hypryou-utils
# yay -S hypryou-greeter
# paru -S hypryou-utils
# paru -S hypryou-greeter

Install the main package first. Add hypryou-utils for GTK-based dialogs and hypryou-greeter if you want greetd integration.

archive

Method 2: Release packages

Best when you want pre-built packages without building from source.

cd <folder-with-.pkg.tar.zst>
sudo pacman -U hypryou-*.pkg.tar.zst
# optional
sudo pacman -U hypryou-utils-*.pkg.tar.zst
sudo pacman -U hypryou-greeter-*.pkg.tar.zst

Download the packages from the GitHub releases page first, then run the commands from the folder where those package files actually live.

terminal

Method 3: Manual build

Use this when you want full control over dependencies and install paths.

git clone --depth=1 https://github.com/koeqaife/hyprland-material-you.git
cd hyprland-material-you
# install dependencies listed in depends.txt
cd hypryou && ./build.sh
cd ../build && ./build.sh

# copy runtime files
sudo cp -r ../hypryou /usr/lib/hypryou
sudo cp -r ../hypryou-assets /usr/share/hypryou
sudo cp hypryouctl hypryou-start hypryou-crash-dialog /usr/bin/
sudo cp ../assets/hypryou.desktop /usr/share/wayland-sessions/

Optional packages can be built with makepkg -si inside hypryou-utils/ and greeter/.

info

If dependency resolution fails, install the missing packages from AUR first. If Hyprland reports a config issue after install, try hyprctl reload before assuming the setup is broken.
